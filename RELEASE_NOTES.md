# v1.0.1

## 功能

- 提供计划睡眠时长、启用状态和闹钟名称配置界面。
- 直接监听 `SleepModelSettings.isStartNow=true`，不依赖手机免打扰状态。
- 按“实际入睡时间 + 计划睡眠时长”创建 OPPO 时钟一次性闹钟。
- 目标时间向上取整到下一整分钟，避免计划时长因秒数被缩短。
- 支持界面中创建 2 分钟后的测试闹钟。

## v1.0.1 修复

- 修复修改计划睡眠时长后仍固定使用默认 8 小时的问题。
- UI 配置改为通过显式配置桥同步到 OPPO 健康 `:transport` 进程。
- 不再依赖 `XSharedPreferences` 跨 UID 读取。

## 兼容性

- OPPO 健康：`6.4.6_cb99e90_260626`
- OPPO 时钟：`16.19.0`
- Android：12 及以上
- LSPosed 作用域：仅 `OPPO 健康`
